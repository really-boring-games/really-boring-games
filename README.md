# Priorities:

**Implement accessible features and inclusive design**

_VR can be incredibly restrictive, often forcing people to play a specific way without providing an alternative means of access. Users must have the agency to choose how they want to interact with the application._

**Accessibility Considerations**

- Cost
- Chronic pain
- Limited vision
- Weight of the headset
- Options for adjusting visual settings
