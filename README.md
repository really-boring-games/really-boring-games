# ***wtf is a QR code?***
Don't panic  👍 
QR Codes are a lot like the barcodes we're all used to.
 - [ ] Open the camera on your phone and hold it like you're going to take a photo.
 - [ ] A button will pop up on your screen that shows the webpage address.
 - [ ] Tap that button and have a look around

[Click through here](https://en.wikipedia.org/wiki/QR_code) to read more about how to make your device do the thing.

# Resources

## [WebGL](https://github.com/KhronosGroup/glTF)
WebGL™ is a cross-platform, royalty-free open web standard for a low-level 3D graphics API based on OpenGL ES, exposed to ECMAScript via the HTML5 Canvas element. Rendering is handled by WebGL
[Learn more](https://www.khronos.org/webgl/) about WebGL

## [glTF™](https://github.com/KhronosGroup/glTF)
GL Transmission Format (glTF) is an open-source and royalty-free 3D file that supports static models, animation, and moving scenes. glTF is used in games, native web application
[Learn more](https://www.khronos.org/webgl/) about KhronosGroup
